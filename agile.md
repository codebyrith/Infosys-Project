# Agile Development Documentation

## 📌 Project Overview
Crypto Investment Manager is a Python-based system designed to help users manage cryptocurrency investments efficiently. The system uses rule-based logic, historical data analysis, and parallel processing techniques to calculate optimal investment mixes, assess risk, and generate reports with alerts.

---

## 👤 User Stories

- As a user, I want to calculate the best mix of crypto assets so that I can maximize returns with minimal risk.
- As a user, I want the system to analyze market trends so that I can make informed investment decisions.
- As a user, I want to receive alerts when significant changes occur so that I can react quickly.
- As a user, I want reports of my investments so that I can track performance.
- As a user, I want the system to automatically adjust investment rules so that my portfolio remains balanced.

## 🔄 System Workflow / Architecture

1. Input crypto data (historical prices)
2. Process data using mathematical and rule-based logic
3. Run risk analysis and prediction (parallel processing)
4. Generate investment suggestions
5. Store and export results (CSV, database)
6. Send alerts for important changes
7. Display insights to the user
